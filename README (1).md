# Breast Cancer Classification using SVM & Gradio Interface

This repository contains a machine learning project that classifies breast cancer tumors as benign or malignant based on cellular features using a Support Vector Machine (SVM) model. The project also includes an interactive web interface built with Gradio.

## 🔗 Quick Links
- [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/drive/1xq9QpbePWMK30qMAZ1n3uh0KPsG-Jq4a?usp=sharing)
- [📁 View Dataset](data.csv)

## 🌟 Features
- **Data Preprocessing & Analysis:** Handled missing data, normalized features, and visualized dataset distributions.
- **Machine Learning Model:** Trained a **Support Vector Machine (SVM)** classifier with optimized hyperparameters.
- **Evaluation:** Achieved high accuracy, detailed in the classification report and confusion matrix within the notebook.
- **Interactive UI:** A user-friendly web interface powered by **Gradio** for real-time predictions.
